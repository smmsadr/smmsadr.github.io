## Context

Behestan Rayan runs the technology that supports its enterprise
operations. My role covered both the Linux systems side and the
database administration for the applications the business ran on.

## Highlights

- Managed Linux servers and the core networked services around them
  — DNS, mail, file services, backups — that kept internal
  operations running.
- Administered Oracle and MySQL database environments: schema
  changes, performance tuning, backup/restore procedures, and
  capacity planning.
- Supported internal application delivery by keeping the platforms
  those apps depended on stable and observable.

## Reflection

Where the discipline of production data set in. Databases don't
forgive shortcuts — every schema decision echoes for years. That
patience carried directly into how I later approached model
pipelines and analytical data platforms.
